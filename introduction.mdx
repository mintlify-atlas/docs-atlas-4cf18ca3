## What is acmcsuf.com?

[acmcsuf.com](https://acmcsuf.com) is the official website for the ACM chapter at California State University, Fullerton (CSUF). It is a fully open-source project built with [SvelteKit](https://kit.svelte.dev/) and deployed on [Cloudflare Workers](https://workers.cloudflare.com/).

The site serves as the digital home for CSUF's ACM community — connecting students with events, workshops, hackathons, blog posts, and team information.

## Tech stack

| Technology | Purpose |
|---|---|
| [SvelteKit](https://kit.svelte.dev/) | Full-stack web framework |
| [Svelte 5](https://svelte.dev/) | UI component library |
| [TypeScript](https://www.typescriptlang.org/) | Type-safe JavaScript |
| [Cloudflare Workers](https://workers.cloudflare.com/) | Edge deployment platform |
| [Vite](https://vitejs.dev/) | Build tooling |
| [Vitest](https://vitest.dev/) | Unit testing |
| [Google Calendar API](https://developers.google.com/calendar) | Events data source |
| [GitHub Discussions](https://docs.github.com/en/discussions) | Blog/newsletter data source |
